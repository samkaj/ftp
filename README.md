# ftp client

A blazingly slow ftp client, written in C++20. Done as a warmup for learning the language, was fun.

## Specs

- Operations
  - [x] `ls`
  - [x] `mkdir`
  - [x] `rm`
  - [x] `rmdir`
  - [x] `cp`
- Interfaces
  - [x] CLI
